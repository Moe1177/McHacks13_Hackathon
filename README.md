# Hackathon Outreach Bot

An all-in-one automated solution for sales, marketing, and sponsorship outreach.

## Features

### Core Functionality
- **Company Discovery** - Input a company name or generate suggestions based on interests/prompts/sectors
- **Email Scraping** - Automatically scrape websites to find email addresses
- **Smart Filtering** - AI-powered filtering to identify personal, HR, and recruiter emails
- **Social Matching** - Cross-reference emails with LinkedIn profiles and other social platforms
- **Profile Verification** - Use LinkedIn data to verify contact information
- **Custom Outreach** - LLM-generated personalized email content

### Frontend
- Scrape by company or by sector
- View all scraped companies with filtering options
- Send custom emails per company
- Filter results by company or sector
- Confidence badges for result accuracy
- Company pipeline view (Discovered → Contacted → Replied)
